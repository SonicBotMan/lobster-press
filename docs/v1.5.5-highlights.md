## 🔥 核心亮点

### 1. 三阶段智能压缩

```
输入对话 → TF-IDF 评分 → Embedding 去重 → 提取式摘要 → 压缩输出
```

**效果：**
- ✅ 保留最重要消息（TF-IDF 三层评分）
- ✅ 移除重复内容（语义相似度 > 0.82）
- ✅ 生成精炼摘要（不生成新 token）

---

### 2. 六大智能模块

| 模块 | 功能 | 价值 |
|------|------|------|
| **TFIDFScorer** | TF-IDF 三层评分 | 精准识别重要消息 |
| **SemanticDeduplicator** | 语义去重 | 移除相似内容 |
| **ExtractiveSummarizer** | 提取式摘要 | 不引入 AI 幻觉 |
| **MessageTypeWeights** | 消息类型权重 | 用户消息优先保留 |
| **ToolResultExtractor** | 工具结果提取 | 压缩长工具输出 |
| **EmbeddingDeduplicator** | Embedding 去重 | 语义级别去重 |

---

### 3. 零 API 成本

**API 调用: 0** - 完全本地化

- ✅ **无幻觉风险** - 不生成新 token
- ✅ **可解释性 100%** - 每步都可追溯
- ✅ **成本归零** - 不调用任何付费 API

## 🚀 快速开始

### 1️⃣ 克隆项目
```bash
git clone https://github.com/SonicBotMan/lobster-press.git
cd lobster-press
```

### 2️⃣ 安装依赖
```bash
pip install -r requirements.txt
```

### 3️⃣ 运行压缩
```bash
# 单个会话
python scripts/lobster_press_v152.py session.jsonl -o compressed.jsonl

# 批量压缩（6.67x 性能提升）
python scripts/batch_compressor.py sessions/ compressed/ --workers auto
```

**完成！** ✅
