### 计算方式

```
净收益 = (原始 Token - 压缩后 Token) - 压缩开销 Token

其中：
- 原始 Token: tiktoken 精确计数（cl100k_base）
- 压缩后 Token: 保留消息 + 摘要的 Token 总数
- 压缩开销 Token: 0（v1.2.0 local 模式）
```

### v1.2.0 改进

## 📊 成本计算器

### v1.2.0（Zero-cost 本地模式）

```bash
# 估算净收益（开销 = 0）
context_tokens=5000
strategy="medium"
overhead=0  # v1.2.0 零成本！

case $strategy in
  light) savings_rate=0.10 ;;
  medium) savings_rate=0.30 ;;
  heavy) savings_rate=0.45 ;;
esac

net_savings=$(echo "$context_tokens * $savings_rate - $overhead" | bc)
echo "预计净收益: $net_savings Token"

# v1.2.0 总是正值
echo "✅ 值得压缩（零 API 成本）"
```

### v1.1.0 对比（API 模式）

```bash
# v1.1.0 需要考虑 API 开销
context_tokens=5000
overhead_v111=1600

net_savings_v111=$(echo "$context_tokens * $savings_rate - $overhead_v111" | bc)

if [ $net_savings_v111 -gt 0 ]; then
  echo "v1.1.0: ✅ 值得压缩"
else
  echo "v1.1.0: ❌ 不建议压缩（负收益）"
  echo "v1.2.0: ✅ 建议升级到 v1.2.0（零成本）"
fi
```
